# PEDAL Prompt: mcp-fastapi-backend

"Python으로 중앙 관리 백엔드를 구축하려고 해. FastAPI를 웹 프레임워크로 사용하고, 그 안에 공식 mcp 라이브러리의 SSE 전송(SSE transport) 방식을 결합해서 하나의 서버로 띄우는 구조를 설계해 줘.
1. Next.js 대시보드와 통신할 기본 REST API 엔드포인트 세팅
2. AI(Cursor, LangChain)와 통신할 /mcp/sse 엔드포인트 세팅
이 두 가지가 공존하는 server.py의 구조와 환경 설정(pyproject.toml 또는 requirements.txt)을 기획해 줘."
